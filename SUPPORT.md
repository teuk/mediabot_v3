# Mediabot v3 support

Thank you for using Mediabot v3.

Please choose the appropriate support channel so that questions and problems can be handled efficiently.

## Report a reproducible bug

Open a **GitHub Issue** and select the **Bug report** template.

Please include:

* your Mediabot version;
* your operating system and Perl version;
* the IRC network concerned;
* clear reproduction steps;
* the expected and actual behavior;
* relevant sanitized logs.

Never publish passwords, API keys, database credentials, IRC passwords, cookies or other private information.

## Suggest an improvement

Open a **GitHub Issue** and select the **Feature request** template for a concrete and clearly described improvement.

For early ideas or proposals that still need discussion, use the **Discussions** tab instead.

## Installation and usage questions

Use **GitHub Discussions** for:

* installation help;
* configuration questions;
* usage questions;
* general troubleshooting;
* ideas that are not yet ready to become feature requests.

## Live IRC community chat

Join **#i/o** on the **EpiKnet** IRC network for live discussion and quick community help.

IRC conversations are useful for diagnosis and discussion, but confirmed bugs should also be recorded in GitHub Issues so they can be tracked.

## Contributions

Documentation fixes, tests, plugins, command improvements and code contributions are welcome.

Contribution guidelines will be available in `CONTRIBUTING.md`.

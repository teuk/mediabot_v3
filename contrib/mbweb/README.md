# mbweb — Mediabot v3 web console

`mbweb` is the Node.js/Express web console for Mediabot v3.

It is not a public project showcase or an installation history page. Its goal is to provide a user-aware web space backed by the Mediabot database.

## Goals

- login with an existing Mediabot user;
- profile view;
- channel visibility according to Mediabot rights;
- radio status from Icecast;
- commands and quotes browsing;
- users/global views for Owner/Master accounts;
- network/metrics views for privileged users.
